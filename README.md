# 🥔 Potato Leaf Disease Detection using CNN

## 📘 Project Description
This project uses a Convolutional Neural Network (CNN) to classify potato leaf diseases from images. The model identifies three classes: Early Blight, Late Blight, and Healthy leaves, helping in early disease detection for better agricultural decisions.

---

## 🎯 Objectives
- Understand image classification using deep learning  
- Perform image preprocessing and data augmentation  
- Build and train a CNN model  
- Evaluate model performance  

---

## 📂 Dataset
- Source: Kaggle  
- Classes:
  - Early Blight  
  - Late Blight  
  - Healthy  
- Type: Image dataset  

---

## 🛠️ Tech Stack
- Python  
- TensorFlow / Keras  
- NumPy  
- Matplotlib  
- Seaborn  

---

## ⚙️ Project Workflow
1. Data Collection  
2. Data Preprocessing (resizing, normalization)  
3. Data Augmentation  
4. Model Building (CNN)  
5. Model Training  
6. Model Evaluation (Accuracy, Confusion Matrix)  

---

## 🧠 Model Architecture
- Convolutional Layers  
- MaxPooling Layers  
- Dense Layers  
- Dropout for regularization
